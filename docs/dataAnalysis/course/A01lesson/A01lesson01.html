木婉清2023/11/30

# 大数据分析背景

  1. 数据和信息是两个相同的概念。

    • 正确
    • 错误
    答案

    错误

  2. 数据的类型主要包括：

    • 文本
    • 图片
    • 音频
    • 视频
    答案

    文本、图片、音频、视频

  3. ____ 已和物质资源、人力资源一样成为国家的重要战略资源，也被称作“未来的石油”。

  • 大气资源
  • 水资源
  • 土地资源
  • 数据资源
答案

数据资源

  1. 大数据来源主要是来自互联网公司、物联网设备、部分企业以及政府部门等的数据资源。

    • 正确
    • 错误
    答案

    正确

# 大数据分析的应用

  1. 大数据最核心的应用之一是：

    • 分析预测
    • 决策制定
    • 技术创新
    • 以上都是
    答案

    分析预测

  2. 大数据应用的重要目的是以下哪一项：

    • 分析预测
    • 决策制定
    • 技术创新
    • 以上都是
    答案

    决策制定

  3. 大数据的应用主要体现的三个方面为：、、____。

    答案

    分析预测、决策制定、技术创新

  4. 以下哪些是大数据的应用：

    • 高能物理
    • 推荐系统
    • 搜索引擎系统
    • 以上都是
    答案

    以上都是

    • 错误
    答案

# 大数据分析工具

  1. 大数据处理框架中最常用的为哪一个：

    • Hadoop 框架
    • Storm 框架
    • Samza 框架
    • Spark 框架
    答案

    Hadoop 框架、Storm 框架、Samza 框架、Spark 框架
